# time-diet-timer

집중력을 키우기 위한 미니멀 타이머입니다.  
25분 공부 + 5분 휴식으로 구성된 **Pomodoro 스타일** 타이머입니다.
필요한 순간에만 알림이 울려 집중을 방해하지 않아요!

---

## 기능

- 25분 집중 + 5분 휴식 루틴 반복
- **딱 5분 남았을 때만** 알림 출력
- 사용자가 원하면 반복 실행
- 콘솔 기반으로 가볍게 실행 가능

---

## 사용 방법

```bash
# 실행 방법
python timer.py
```
